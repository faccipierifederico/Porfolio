Proyectos terminados

Los dos proyectos que se presentan a continuación fueron realizados bajo el marco del Bootcamp Front-End de Argentina Programa y bajo la tutela de mis mentores de MindHub.

"Mighty Ducks Hockey League" se trató del primero de ellos, en donde el cliente solicitaba una web con diversos htmls donde cada uno tenía diferentes funcionalidades. La idea general fue realizar el proyecto a través de metodologías ágiles, donde debíamos cumplir plazos de entrega con avances del proyecto así como informes diarios a través de las dailys con nuestros mentores.

Tanto la maquetación de la web como los estilos son personales, sin embargo debíamos cumplir con ciertas pautas que venían como directivas por parte del cliente.

Visitar el sitio web

"Amazing Events" se trata de un proyecto mucho más ambicioso, en donde las herramientas que debíamos aplicar son más avanzadas. En este debimos utilizar bootstrap, así como también estilos de nuestra propia impronta, además de JavaScript. Este lenguaje fue utilizado con tres grandes objetivos: el primero de ellos se basó en poder realizar filtros a través de checkboxes y así lograr filtrar los diferentes tipos de eventos que se presentaban en la web por sus respectivas categorías; el segundo se trataba también de un filtro, pero en este caso a través de una barra de búsqueda. El último objetivo consistía en, a través de JS, conectarse con una API y traer toda la información necesaria.

Finalmente también fue realizado todo este proyecto bajo el framework de VUE.js
